# Fabric data generation

This directory contains scripts and collection definitions used to generate or describe the preload datasets used by the Fabric benchmarks.

## Contents
- `generate_dummy_data.py`: synthetic candidate and voter data.
- `booth_pdc_compact_uttar_pradesh.py`: compact booth preload generator.
- `collections-voter-roll.json`: private-data collection configuration.

## Notes
- Keep generated CSV or JSON outputs outside versioned source unless they are canonical fixtures.
