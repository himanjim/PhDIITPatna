# Frontend performance artefacts

This directory stores the inputs and outputs used by the frontend performance-measurement scripts.

## Contents
- `constituencies.txt`: input fixture.
- `api_measurements.csv`, `summary.txt`: API measurement outputs.
- `bundle_sizes.json`, `bundle_sizes.md`: bundle-size reports.
- `lighthouse.html`: Lighthouse output.

## Notes
- Most files here are generated artefacts. Keep them only if they are intentionally versioned for reporting.
