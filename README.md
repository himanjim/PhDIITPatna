# PhDIITPatna

This repository contains two major components:

## 1) /ai
AI modules used in the project (face verification, FAISS de-duplication, Triton inference benchmarking, liveness checks, and supporting scripts).

## 2) /fabric
Hyperledger Fabric v3.1.1 blockchain module for internet voting benchmarks:
- RAFT and SmartBFT network configs
- Chaincodes (accumvote, boothpdc, evote-preload)
- Benchmark harnesses (Caliper + Node.js)
- Results and supporting docs
