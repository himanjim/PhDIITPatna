# Documentation and Runbooks

This directory stores setup notes, command runbooks, and experiment-support documents for the AI component.

## Contents
- `insight_faiss_pipeline_setup.txt` records the setup flow for the InsightFace-plus-FAISS pipeline.
- `faiss_tests_commands.docx` contains Windows command sequences for FAISS service setup, tests, and benchmarking.
- `lvs_commands.docx` contains Windows command sequences for the liveness-verification service.
- `grpc_triton_faiss_commands.docx` records the FAISS gRPC and Triton benchmark runbook.
- `lvs_a100_commands.docx` records the Linux A100 liveness-service benchmark runbook.

## Scope
This directory is for human-facing operational material. New runbooks, setup notes, or benchmark execution guides should be added here rather than placed inside code directories.
